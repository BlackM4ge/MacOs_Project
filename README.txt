Ecco il programma per lo scanner.

Scarica il file zip e apri la cartella.

Trascina l'icona AI_Scanner nella tua cartella Applicazioni.

IMPORTANTE (Solo la prima volta): Non fare doppio clic. Fai Tasto Destro sull'icona -> Apri -> e poi clicca Apri nel messaggio di avviso.

Come funziona:

Clicca su "Carica Foto".

Controlla i numeri che appaiono nella casella di testo.

Clicca su "Salva su Excel".

Troverai automaticamente un file chiamato Report_Numeri_AI.xlsx sulla tua Scrivania.
